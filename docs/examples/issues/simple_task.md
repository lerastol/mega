# Простая задача (один цикл, ожидается APPROVE)

**Цель:** проверить, что Code Agent создаёт PR, Reviewer одобряет без правок.

## Пример формулировки для Issue

```
Исправь опечатку в README: замени слово "примвер" на "пример" в первом абзаце.
```

Или:

```
Добавь в requirements.txt библиотеку python-dotenv (если её ещё нет).
```

## Ожидаемое поведение

1. Code Agent создаёт ветку `fix/issue-<N>`, вносит изменение, запускает проверки (black, ruff, mypy, pytest), пушит и создаёт PR с «Closes #N».
2. AI Reviewer анализирует PR и выносит **APPROVE** (изменения минимальны и соответствуют задаче).
3. Метки: **reviewed**, снята **ai-thinking**. Цикл правок не требуется.
