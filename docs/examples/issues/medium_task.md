# Задача с доработкой (1–2 итерации после замечаний)

**Цель:** проверить цикл Reviewer → REQUEST_CHANGES → Code Agent Fix → повторное ревью → APPROVE.

## Пример формулировки для Issue

```
Добавь в проект функцию, которая по пути к файлу возвращает его размер в байтах.
- Функция: get_file_size(path: str) -> int
- Размести в src/utils.py (создай файл, если его нет)
- Добавь тесты в tests/test_utils.py: файл существует / файл не существует (ожидаем ошибку или 0 по договорённости)
```

Или:

```
В README добавь раздел "Локальный запуск" с шагами: создание venv, активация, pip install -r requirements.txt, пример команды python src/main.py --issue 1
```

## Ожидаемое поведение

1. Code Agent создаёт/меняет файлы, создаёт PR.
2. Reviewer может запросить правки (например, стиль, граничные случаи, оформление README).
3. После REQUEST_CHANGES срабатывает Code Agent Fix: правки по замечаниям, push в ту же ветку.
4. Повторное ревью → **APPROVE** (возможно после 1–2 итераций).
5. В теле PR виден рост счётчика итераций `<!-- iteration: N -->`.
